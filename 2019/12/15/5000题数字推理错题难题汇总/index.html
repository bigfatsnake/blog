5000题数字推理错题难题

  1. （2012 浙江 43）243，162，108，72，48，( )
    A.26 B.28 C.30 D.32
    【解析】观察数字，有很好的因式分解性，243=381，162=281，108=618，72=324，48=2*2，容易发现243/162=3/2，72/48=3/2，验证其他两项符合前项÷后项=3/2，即成等比数列，公比为3/2。所以，( )=48÷3/2=32。

  2. * （2012 江苏 C 21）4，6，10，14，22，( )
    A.24 B.26 C.32 D.38
    【解析】观察数字，都有因数2，分出后剩余因数是2，3，5，7，11，（13）成质数数列，所以( )=2*13=26。

  3. （2013 浙江 B 41）2，2，7，9，16，20，( )
    A.28 B.29 C.30 D.31
    【解析】观察数字，整体变化趋缓，考虑作差。作差后无规律，考虑数字较小，作和。先两两作和，得到4，9，16，25，36，（49）成次幂数列，( )=49-20=29。

  4. （2015 广东 22）2，10，30，68，130，( )
    A.222 B.272 C.300 D.390
    【解析】方法一：观察数字，整体变化趋缓，考虑作差。作差后，得到8，20，38，62，（92），规律依然不明显。再次作差，得到12，18，24，（30）成等差数列，公差为6。所以，( )=30+62+130=222。方法二：观察数字，均为次幂数附近数字，尝试凑修正次幂，得13+1=2，23+2=10，33+3=30，43+4=68，53+5=130，所以( )=63+6=222。

  5. （2013 浙江 B 42）1，-3，3，3，9，( )
    A.28 B.36 C.45 D.52
    【解析】观察数字，整数性良好，作商。作商后，得到-3，-1，1，3，（5）成等差数列，公差为2。所以，( )=9*5=45。

  6. （2013 天津 4）3，2，11，14，27，( )
    A.32 B.34 C.36 D.40
    【解析】方法一：观察数字，整体变化趋缓，考虑作差。作差后，得到-1，9，3，13，（5），规律依然不明显。再次作差，得到10，-6，10，（-6）成周期数列。( )=-6+13+27=34。方法二：观察数字，作差会单调性混乱，考虑数字较小，作和。先两两作和，得到5，13，25，41，( )，规律依然不明显。再次作和，得到18，38，66，( )规律依然不明显。那么第二次不作和，作差。得到8，12，16，（20），成等差数列，公差为4。( )=20+41-27=34。

  7. （2012 江苏 B 84）5，6，( )，10，15，30
    A.7 B.9 C.7+1/2 D.9+1/2
    【解析】观察数字，整数性良好（一般以上的项数具备即可），作商。作商后，得到6/5，( )，( )，3/2，2。凑5/4，4/3，选C。

  8. （2015 吉林乙 51）5，9，11，17，( )
    A.21 B.22 C.26 D.30
    【解析】方法一：观察数字，整体变化趋缓，考虑作差。作差后，得到4，2，6，( )规律不明显。考虑到单调性混乱，数字较小，作和。得到6，8，( )。因为给出的项数少，开始凑，比如凑10，成等差数列，公差为2。那么，( )=10-6+17=21。方法一：观察数字，数字较小，考虑作和。作和后得，14，20，28，( )规律不明显，继续作和。发现无规律，那么换做差，得6，8，( )。因为给出的项数少，开始凑，比如凑10，成等差数列，公差为2。那么，( )=10+28-17=21。

  9. * （2015 吉林甲 89）3，5，7，11，19，( )，307
    A.23 B.57 C.51 D.37
    【解析】观察数字，整体变化趋缓，考虑作差。作差后，得到2，2，4，8，( )，( )，这组数列的规律太多，考虑到最后一项数字陡增，找次幂和乘积关系。观察得到22=4，24=8，所以后两项为48=32，832=256。带入验证，符合。所以，( )=19+32=51=307-256。

  10. （2017 浙江 B 10）-1，1，3，10，19，( )，55
    A.27 B.35 C.43 D.56
    【解析】观察数字，整体变化趋缓，考虑作差。作差后，得到2，2，7，9，( )，( )，规律不明显。再次观察数字，数字较小，考虑作和。作和后得，0，4，13，29，( )，( )，虽然规律不明显，但是看到万能数字4，很惊喜。观察只要作差，可以出现平方数数列，即4，9，16，（25），（36）。所以，( )=25+29-19=35=29+25+36-55。

  11. （2013 陕西 73）13，16，20，34，92，( )
    A.1124 B.906 C.1316 D.832
    【解析】观察数字，整体变化趋缓，考虑作差。作差后，得到3，4，14，58，( )，为基础递推，34+2=14，414+2=58，1458+2=？以4结尾，+92，( )以6结尾，只能排除两个选项，精确计算。1458+2+92=580+232+94，一千以内，选B。

  12. （2011 江苏 B 83）-30，-4，( )，24，122，340
    A.-1 B.-2 C.6 D.13
    【解析】观察数字，有次幂数附近数字，凑次幂。340=73-3，122=53-3，24=33-3，（-2）=13-3，-4=（-1）3-3，-30=（-3）3-3。【注】122附近有125、121、128等次幂数，扩散思维。

  13. （2011 江苏 C 23）-64，4，0，1，1/4，( ）
    A.8 B.64 C.1/16 D.1/36
    【解析】观察数字，有次幂数附近数字，凑次幂。1/4=4-1，1=20，0=01，4=（-2）2，-64=（-4）3，所以（）=6-2。

  14. （2017 江苏 A 57）4，5，7，16，80，( )
    A.296 B.423 C.592 D.705
    【解析】观察数字，整体变化趋缓，考虑作差。作差后，得到1，2，9，64，（），出现次幂数，凑次幂。得到1=10，2=21，9=32，64=43，后一位为54。所以（）是5结尾，选D。

  15. （2015 广东 24）2/5，3/10，7/30，23/210，( )
    A.31/967 B.35/1208 C.159/2282 D.187/4830
    【解析】观察分母，成倍数变化，分别是2，3，7，（），跟分子相同，所以（）的分母为210*23，选D。备注：分子为前一项的分母-分子。【技巧】从明显特征入手。

  16. * （2016 浙江 B 30）0，1，3/2，11/6，25/12，( )
    A.137/30 B.137/60 C.137/90 D.137/100
    【解析】方法一：观察分母，成倍数关系，通分化一成60，分子为0，60，90，110，125。作差后得，60，30，20，15。都有5，作商得，1/2，2/3，3/4，（4/5，）。所以，最后一项分子为125+12=137，结合分母，选B。方法二：无法凑单调性，作差。作差后，得到1，1/2，1/3，1/4，（1/5），分子为1，分母为自然数列。所以，（）=25/12+1/5=137/60。

  17. * （2013 广东 4）1，5/6，7/10，3/5，8/15，( )
    A.1/5 B.1/4 C.1/3 D.1/2
    【解析】方法一：观察分母，成倍数关系，通分化一成30，分子为30，25，21，18，16，（）。作差后得，5，4，3，2，（1），成等差数列。所以，最后一项分子为16-1=15，结合分母，选D。方法二：无法凑单调性，作差。作差后，得到1/6，2/15，1/10，1/15，（），看不出规律，通分分母化一成30，分子分别为5，4，3，2，（1）。所以，（）=8/15-1/30=1/2。

  18. （2013 浙江 B 45）3/2，1/2，1/4，3/20，1/10，( )
    A.1/14 B.1/15 C.1/16 D.1/17
    【解析】观察分子，出现多个1，分子化同。分子化为3，则分母为2，6，12，20，30，（），规律不明显，作差。作差可得，4，6，8，10，（12）为等差数列。所以，（）的分母=30+12=42，结合分子3，选A。

  19. * （2014 江苏 C 25）1/4，2/3，14/9，28/9，140/27，( )
    A.280/27 B.560/27 C.280/81 D.560/81
    【解析】观察数字，分子分母出现很好的整除性，作商。作商后得，8/3，7/3，2，5/3，（）成等差数列，公差为1/3。所以，（）=（140/27）*（5/3-1/3）=560/81。

  20. * （2013 天津 5）3，-15/4，14/5，-45/28，( )
    A.25/36 B.33/41 C.21/48 D.35/64
    【解析】观察数字，没有整除性，分子分母没有倍数关系，反约分。14/5=28/10，则分母分别为（），4，10，28，（）。分母作差得，（2），6，18，（54）。出现3倍关系，前后尝试凑3倍，则分子为，6，15，28，45，（）。作差得9，13，17，（21）成等差数列，所以（）=（45+21）/（54+28）=33/41。

  21. * （2012 吉林甲 3）19，7，37/9，23/8，( )
    A.19/6 B.16/9 C.17/7 D.11/5
    【解析】观察数字，出现幂次数，凑幂次。19/12，28/22，37/32，46/42，（），分母成平方数列，分子成等差数列，公差为9。所以，（）=55/52=11/5。

  22. * （2013 浙江 A 44）6，3，5，13，2，63，( )
    A.-36 B.-37 C.-38 D.-39
    【解析】观察数字，变化趋势混乱。本题看四项，前两项之积=后两项之和。63=5+13，35=13+2，13*2=63+（），所以（）=26-63=-37。

  23. * （2015 浙江 A 52）5，11，-3，7，-5，( )
    A.6 B.7 C.8 D.9
    【解析】观察数字，变化趋势混乱。本题看三项，前两项之差÷系数=第三项。（5-11）÷2=-3，（11+3）÷2=7，（-3-7）÷2=-5，（7+5）÷2=（）=6。

  24. （2017 浙江 B 5）80，56，52，30，37，( )
    A.21/2 B.11 C.23/2 D.12
    【解析】观察数字，变化趋势平缓，倍数关系，选项分母出现2，凑2倍关系。52=80-56/2，30=56-52/2，37=52-30/2，（）=30-37/2=23/2。

  25. * （2013 浙江 A 39）3，7，12，15，9，-18，( )
    A.-27 B.-45 C.-81 D.-102
    【解析】观察数字，出现负号，有减法运算；出现公因数3，有3倍关系。本题递推关系为前两项作差3=第三项。（）=（-18-9）3=-81。

  26. （2012 江苏 C 23）2，3，7，( )，121，721
    A.11 B.17 C.19 D.25
    【解析】观察数字，最大两项关系在6倍左右，入手拆开。1216-5=721，（25）5-4=121，74-3=25，33-2=7，2*2-1=3。

  1. * （2017 新疆 60）2，3，4，15，56，( )
    A.285 B.235 C.245 D.225
    【解析】观察数字，选项均有因数5。56=154-4，15=43+3，4=32-2，3=21+1，（）=56*5+5=285。

  2. * （2017 江苏 B 58），3-，2，3，4+，（）
    A.5+2 B.6+5 C.7+3 D.9+6
    【解析】观察数字，无特征，作差。得3-2，-1，1，，（），第一项+2*第二项=第三项。所以，选C。

  3. * （2013 河北 36）1，3，6，9，9，( )
    A.0 B.6 C.9 D.18
    【解析】观察数字，有3这个因数，围绕3凑。前两项之差*3=第三项。选A。

  4. * （2011 江苏 A 20）1，1，2，( )，16，625
    A.3 B.5 C.7 D.9
    【解析】观察数字，项数少，交叉分组，考虑两组的共性。奇数项为2的0，1，4，偶数项为5的0，1，4，选B。

  5. （2015 河北 59）-2-5，8，9，-14，-13，20，17，-26，( )
    A.-21 B.21 C.-29 D.29
    【解析】观察数字，项数多，分组。隔两项出现负号，两两分。组内作差，得3，1，-1，-3，（-5），所以（）=-21。
    【注】本题负号决定答案是否含有负号，计算时不带负号。

  6. * （2012 广东 4）2，2，8，-1，-2，5，1，1，2，-1，1，( )
    A.-2 B.-1 C.1 D.2
    【解析】观察数字，项数居多，三三分组。组内成平方和，所以（）=2。

  7. （2011 江苏 B 84）23，57，1113，1317，( )，2331
    A.1921 B.1715 C.1723 D.2129
    【解析】观察数字，中分后为质数数列。

  8. （2017 江苏 A 58）4.1，4.3，12.1，12.11，132.1，( )
    A.120.8 B.124.12 C.132.131 D.132.12
    【解析】观察数字，整数部分和小数部分出现倍数关系，考虑运算。前一项整数部分*小数部分=后一项的整数部分，前一项整数部分-小数部分=后一项的小数部分。所以，（）=132.131。

  9. （2013 江苏 A 21）12，23，35，47，511，( )
    A.613 B.612 C.611 D.610
    【解析】观察选项，首位皆为6，考虑机械拆分。拆成2段，后段为2，3，5，7，11，（13），质数列。选A。

  10. * （2011 江苏 C 24）12.5，23.4，31.2，( )，52.3，62.3
    A.41.2 B.42.7 C.46.4 D.49.5
    【解析】方法一：观察数字，首位成正整数数列。分出首位得，2.5，3.4，1.2，（），2.3，2.3，两两一组，整数+小数的数字相等。选A。方法二：所有项数字相加得，8，9，6，（7），10，11。两两分组，组内差一。选A。

  11. （2011 江苏 B 82）102，213，316，4310，518，( )
    A.6414 B.6314 C.714 D.1019.29
    【解析】观察数字，中间出现4位数，分两段。为了保持单调想，三位数分成前2位和后1位。得到前面数列为10，21，31，43，51，（64），规律不明显，作差。得到11，10，12，8，（13），奇数项等差。后面数列为2，3，6，10，18，（14），从第二项起，每一项为题干数列各位数字之和。选A。

  12. * （2012 江苏 A 19）212，424，234，446，658，( )
    A.245 B.267 C.233 D.212
    【解析】数字较大，位数工整，机械拆分。分段后，观察中间为自然数列。选B。

  13. （2015 吉林乙 85）1，3，-1，7，21，53，( )
    A.135 B.142 C.169 D.175
    【解析】观察数字，变化平缓，首先作差，无规律，考虑作和。两两作和，无规律。三三作和，得到3，9，27，81，（243）成等比数列。所以，（）=243-21-53=169。

  14. （2011 江苏 C 22）4，12，24，36，50，( )
    A.64 B.72 C.86 D.98
    【解析】观察数字，变化平缓，首先作差。得到8，12，12，14，（），无规律。再次观察数字，有很好的分解性。依次分解为41，62，83，105，（12*6），左边为合数数列，右边为等差数列。选B。
